# VelaBeam

AI-powered automated website builder for local businesses. Find prospects, generate sites, deploy — all from one dashboard.

## Getting Started

```bash
npm install
cp .env.example .env.local
# Edit .env.local with your database URL and auth secret
npm run dev
```

## Environment Variables

See `.env.example` for all required and optional variables.

## Tech Stack

- Next.js 15 (App Router)
- NextAuth.js v5 (Credentials provider)
- PostgreSQL
- Stripe (optional billing)
- Tailwind CSS

## Design System

Premium Studio archetype with charcoal, ivory, muted gold, burgundy palette. Canela-style serif + Neue Haas-style sans typography.
